# The MuffinTime Modpack

![The MuffinTime Minecraft](https://cdn.modrinth.com/data/cached_images/1927b556518ad89576aa2ad6c185d649b2683991.png)

This modpack is designed and used for [The MuffinTime Minecraft](https://modrinth.com/server/muffintime-minecraft).

It contains for one the clientside modpack, and for the other all the mods used on the server side.

Included on clientside are some performance mods and QOL-mods, like mini-/worldmap and voicechat.

Included on serverside are also performance mods and also QOL-mods, like voicechat, BlueMap, F-Sit and player statistics.
